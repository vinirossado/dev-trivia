📚 Table Documentation

👤

User Management

👤 Users

Central table storing user account information and profiles.

Name User's display name
Email User's email for authentication and notifications
PasswordHash Encrypted password (null for OAuth users)
ProfileImageUrl URL to user's profile picture
AuthProvider Authentication method (local, google, github, microsoft)
ExternalId OAuth provider user ID (null for local auth)
Bio User biography or description
Location User's geographic location
DateOfBirth User's date of birth
LastLoginAt Most recent login timestamp
PreferredLanguage User's preferred UI language
🔐 AuthSessions

Manages authentication sessions and refresh tokens.

RefreshTokenHash Hashed token for session renewal
ExpiresAt Session expiration timestamp
RevokedAt Session invalidation timestamp
🎭 UserRoles

Defines user permissions and access levels.

Role Permission level (admin, player, moderator)
📊 PlayerStats

Tracks player performance metrics and skill rating.

TotalMatches Total number of games played
TotalCorrect Cumulative correct answers
EloRating Skill-based ranking score
👁️ QuestionExposure

Prevents question repetition by tracking views.

LastSeenAt Most recent encounter timestamp
TimesShown Question appearance counter
❓

Question Bank

🗂️ Categories

Organizes questions into thematic groups.

Name Category display name
Description Detailed category scope explanation
❓ Questions

Core trivia content with problem statements.

Title Brief question summary
Description Full question text with context
Difficulty Challenge level (Easy, Medium, Hard)
CategoryId Links question to its category
📝 AnswerOptions

Multiple choice answers for each question.

Text Answer choice content
IsCorrect Marks the correct answer
🎮

Game Sessions

🎮 Matches

Represents individual game sessions with metadata.

StartedAt Game initiation timestamp
EndedAt Game completion timestamp
Status Current state (in_progress, completed)
SelectedCategoryId Category chosen for this match
✅ PlayerAnswers

Records every answer for scoring and analytics.

MatchId Links answer to game session
UserId Identifies who submitted the answer
QuestionId References the question being answered
AnswerOptionId Selected answer choice
IsCorrect Cached correctness for quick scoring
AnsweredAt Response time tracking timestamp
